BBEdit Language Module

FortiGate syntax highlighting.
For BBEdit.

A codeless language module that brings full syntax highlighting, keyword coloring, and function navigation to FortiGate configuration files. Built from the FortiOS 7.6 CLI reference.

ℹ️ Converted from Notepad++. This module is a port of the Notepad++ FortiGate user-defined language, expanded with every top-level config section and CLI command from the FortiOS 7.6 reference.
Install
mkdir -p ~/Library/Application\ Support/BBEdit/Language\ Modules
cp FortiGate.plist ~/Library/Application\ Support/BBEdit/Language\ Modules/
Restart BBEdit after copying. Files with .conf extension will automatically use FortiGate highlighting.

What it highlights

Open any FortiGate config backup or snippet and get instant visual structure.

🎨

Keyword Coloring

All FortiOS 7.6 config sections, CLI verbs, sub-objects, and common values colored by role.

📐

Structure Commands

config, edit, set, end, next shown in a distinct predefined-name color.

🧭

Function Navigation

Jump between config and edit blocks using BBEdit's function popup.

💬

Comment Highlighting

Lines starting with # are recognized as comments.

📝

String Delimiters

Double-quoted and single-quoted strings highlighted as string literals.

🔤

Case Insensitive

Matching is case-insensitive, consistent with FortiOS CLI behavior.

Keyword coverage

Built from the FortiOS 7.6 CLI reference. Eight categories of keywords are recognized.

CLI Verbs config, edit, set, unset, get, show, execute, diagnose, delete, abort, append, clone, move, purge, rename
Config Sections All 34 top-level trees: system, firewall, router, vpn, user, ips, dlp, antivirus, webfilter, application, voip, waf, wanopt, log, and more
Sub-objects policy, address, interface, zone, phase1-interface, sd-wan, ssl-vpn, ha, ospf, bgp, and 50+ more
Auth & Identity ldap, radius, saml, tacacs+, local, group, accprofile, api-user, fortiguard, fortimanager
Common Values enable, disable, accept, deny, drop, reject, all, any, none, up, down, transparent
Execute Commands ping, traceroute, shutdown, reboot, backup, restore, factoryreset
Diagnose Commands sniffer, packet, debug, netlink, hardware, test
Cloud & Integration sdn-connector, aws, azure, gcp, oci, nsxt, forticloud, fortianalyzer

Quick start

From download to highlighted config in under a minute.

# 1. Create the Language Modules directory if it doesn't exist
mkdir -p ~/Library/Application\ Support/BBEdit/Language\ Modules

# 2. Copy the plist into place
cp FortiGate.plist ~/Library/Application\ Support/BBEdit/Language\ Modules/

# 3. Restart BBEdit
#    Open any .conf file and you'll see FortiGate syntax highlighting.

# 4. If .conf conflicts with another language module,
#    set the language manually via the status bar popup
#    or rename your files to .fgt

Origin

This module was converted from a Notepad++ user-defined language file for FortiGate configuration syntax. The original covered basic keywords from an older FortiOS release.

The BBEdit port expands coverage to the full FortiOS 7.6 CLI reference, adding every top-level config section, all CLI command verbs, common sub-objects, authentication backends, cloud integrations, SD-WAN, automation, and diagnostic commands.
